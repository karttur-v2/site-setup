    Install Jekyll

    Jekyll

    Jekyll is a web site site generator based on [Ruby][]. Jekyll generates static web pages from text written in markup languages, for example [markdown][markdown]. To generate the layout of the static web pages Jekyll depends on a theme (the topic of the next post). This post focuses on installing and updating Jekyll as a gem under [Ruby][].

    How to install Jekyll as a gem under Ruby is explained in detail on the [Jekyll site][https://jekyllrb.com/docs/installation/macos/].

    First time installation

    If this is your first installation of Jekyll, just go ahead and execute the terminal command:

    gem install jekyll

    This will install the latest version of Jekyll, at time of writing this in November 2025 that is 4.4.1.

    If the terminal prompts ends with something like

    A new release of RubyGems is available: 3.6.9 -> 3.7.2:
Run 'gem update --system 3.7.2' to update your installation.

    just go ahead and follow the suggestion.

    Check Jekyll installation

    To check your installed Jekyll version, run the command

    % jekyll -v
jekyll 4.4.1

    To find where Jekyll is residing in your machine, run the command:

    ``` % which jekyll /Users/thomasgumbricht/.gem/ruby/3.4.2/bin/jekyll
